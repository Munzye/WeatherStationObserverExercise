# Weather Station - Observer Pattern

Proyecto en Java que implementa el **patrón de diseño Observer** para simular una estación meteorológica.

## Descripción
El sistema permite que distintos displays se suscriban a los cambios del clima y se actualicen automáticamente cuando las mediciones cambian.

## Componentes principales
- **Subject:** WeatherData
- **Observers:** Displays meteorológicos
- Current Conditions Display
- Statistics Display
- Forecast Display
- Pressure Display
- Heat Index Display
